## 👥 Add Members to Website

Edit `data.json` file:

```json
{
  "members": [
    {
      "id": 1,
      "name": "Nama Lengkap",
      "role": "Role (e.g. Admin, Member)",
      "bio": "Bio singkat tentang mereka",
      "avatar": "URL foto profil (optional)",
      "social": {
        "whatsapp": "Nomor WA (optional)",
        "instagram": "@username (optional)",
        "twitter": "@username (optional)"
      }
    }
  ]
}
```

**Atau kasih tau saya nama-nama member**, saya yang inputin! 🚀

## 🎨 Customization

### **Change Colors**
Edit CSS variables in `index.html`:
```css
:root {
    --primary: #1DB954;    /* WhatsApp green */
    --secondary: #191414;  /* Dark */
    --accent: #FF6B6B;     /* Accent color */
}
```

### **Change Stats**
Edit in `index.html`:
```html
<div class="stat-number" id="memberCount">1,100+</div>
```

### **Add More Features**
Copy `feature-card` div and customize!

## 🚀 Quick Deploy to GitHub Pages

1. Create GitHub repo
2. Upload `index.html` and `data.json`
3. Go to Settings → Pages
4. Enable GitHub Pages
5. Done! Website live at: `username.github.io/repo-name`
